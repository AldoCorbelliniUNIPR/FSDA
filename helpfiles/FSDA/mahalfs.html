mahalFS

Computes Mahalanobis distances (in squared units) from a prespecified centroid MU and scatter matrix SIGMA

Syntax

d = mahalFS(Y,MU,SIGMA)

Description

d = mahalFS(Y,MU,SIGMA) returns the Mahalanobis distance (in squared units) of each observation (point) in Y using centroid MU and scatter matrix SIGMA.

d(i) = (Y(i,:)-MU) × SIGMA^(-1) × (Y(i,:)-MU)'

i=1, 2, ..., size(Y,1)

Examples

Y=randn(10,2);
MU=median(Y); 
SIGMA=[0.3 0.4; 0.4 1];
% Compute MD using as centroids the medians and shape matrix SIGMA
d=mahalFS(Y,MU,SIGMA);
