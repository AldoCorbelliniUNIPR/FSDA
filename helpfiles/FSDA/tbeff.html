TBeff

Compute the value of the constant c which we have to put in  œÅ function for Tukey biweight to obtain a prefixed location or scale efficiency

Syntax

c = TBeff(eff,p)
c = TBeff(eff,p,1)

Description

c = TBeff(eff,p) calculates value of constant c which is necessary to obtain a prefixed location efficiency. eff is a scalar containing desired level of efficiency. p is a scalar which denotes the number of response variables present in the dataset.

c = TBeff(eff,p,1) calculates value of constant c which is necessary to obtain a prefixed shape efficiency. eff is a scalar containing desired level of efficiency. p is a scalar which denotes the number of response variables present in the dataset.

Examples

To obtain a nominal location efficiency of 0.95 in regression we have to se c equal to 4.685064948559557 
c=TBbdp(0.95,1)
To obtain a nominal shape efficiency of 0.95 in regression we have to se c equal to 5.490249208447494 
c=TBeff(0.95,1,1)
