FSColors

A MATLAB class encapsulating some common color definitions

Syntax

Usage for a given colorname (e.g. colorname=reddish):

RGB_vector = FSColors.colorname.RGB

short_name = FSColors.colorname.ShortName
colorname specifications currently include the standard MATLAB colors

yellow, magenta, cyan, red, green, blue, white, black

and the following newly defined colors:

blueish, reddish, greenish, purplish, yellowish, greysh, lightblue.

Description

FSColors is a MATLAB class encapsulating some common color definitions used in the FSDA toolbox. The associated method returns a cell formed by the RGB vector of the color and a short name for it. See ColorSpec for color specifications in terms of RGB vectors and short names.

    Note   FSColors represents a first test in view of a more extensive use of MATLAB classes/methods constructs in FSDA.
