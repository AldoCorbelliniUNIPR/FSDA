Dynamic Statistical Visualization

add2spm Adds objects to the scatterplot matrix
cdsplot Candlestick plot for robust model selection.
covplot Trajectories of the covariance (correlation) matrix monitored along the search
fanplot Forward plot of the monitoring of the score test statistic for transformation for different values of the Box Cox transformation parameter.
histFS Creates a histogram with the elements in each bin grouped according to a vector of labels.
levfwdplot Plots the trajectories of leverage along the search
malfwdplot Plots the trajectories of Mahalanobis distances along the search
malindexplot Plots the Mahalanobis distances against index number or any other variable.
mdrplot Plots the trajectory of minimum deletion residual (mdr) along the search
mmdplot Plots the trajectory of minimum Mahalanobis distance (mmd) along the search
olsline Add least-squares fit line(s) to scatter plot(s).
resfwdplot Plots the trajectories of scaled (squared) residuals along the search.
resindexplot Plots the residuals against index number or any other variable.
selectdataFS Graphical selection of data points using the mouse, on scatter plot matrices or trajectories in the forward plots.
spmplot Scatterplot matrix with dynamic scatters and histograms
yXplot Generates the scatterplot of y against each variable of X in the input dataset.
