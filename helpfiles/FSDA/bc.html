bc

Binomial Coefficient

Syntax

bc(n,k)
[out] = bc(n,k)

Description

bc(n,k) returns n!/k!(n-k)! i.e. the number of combinations of n things taken k at a time.

This function is a simplified version of the MATLAB function nchoosek. It is faster because all controls on n and k are suppressed. This function was introduced in the FSDA toolbox only for performance reasons. For computing the binomial coefficient in more general contexts we suggest using the standard nchoosek MATLAB function.

    Note   Like for nchoosek, when a coefficient is large results may be inexact. The result is only accurate to 15 digits for double-precision inputs in 32bits computers.
